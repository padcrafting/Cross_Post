# Cross_Post
A wrapper script for posting using Bluesky and Mastodon native CLI clients.

## Pre-requisites:
- install the bluesky ('bsky') and mastodon ('toot')  CLI clients, and login. The script assumes that they are operational.
- requires Python 3.x but no other dependencies
- uses nano as an optional editor
- on first run, it checks for these requirements, and reports if they aren't there. 

## Usage
`python cross_post.py --help`
